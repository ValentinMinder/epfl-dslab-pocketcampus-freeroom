• Emails EPFL
    PocketCampus peut automatiquement configurer l'app Mail pour y recevoir vos emails EPFL.
    Allez dans les réglages PocketCampus depuis le menu principal, puis Email EPFL.
  • VPN EPFL
    PocketCampus peut automatiquement configurer le VPN de l'EPFL sur votre iPhone ou iPad.
    Allez dans les réglages PocketCampus depuis le menu principal, puis VPN EPFL.
